Stock Market Summary
This VBA script is designed to analyze stock market data within an Excel workbook. The sscript begins by declaring respective variables to store information such as ticker symbols, open and close prices, yearly changes, percent changes, and total stock volumes. The script also adds colums headers for the respctive variables. 

Data analysis: 

the Script determines the yearly change, percent change, and total stock volume for each ticker symbol and identifies  opening and closing prices, calculates the changes.

SUmmary Table :

The script creates a summary table for each worksheet, showing the stock with the greatest percent increase, greatest percent decrease, and greatest total volume. 

Use:
After oepning VBA editor press"Alt +F11" then paste the provided script into a new module and run "StockMarket_summary"
